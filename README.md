# ERP-System-for-Students.
# 🎓 Student ERP System (Frontend)

A modern **Student ERP (Enterprise Resource Planning) System** designed for Class 10 students.
This project focuses on a clean, professional UI with essential student features like attendance tracking, timetable, marks, and homework management.

---

## ✨ Features

* 📊 **Dashboard** – Overview of attendance, homework, and exams
* 📅 **Timetable** – Weekly class schedule
* 📌 **Attendance** – Day-wise attendance tracking
* 📈 **Marks** – Subject-wise performance with grading
* 📝 **Homework** – Tasks with due dates and status
* 👤 **User Profile UI** – Top-right profile section
* 🔍 **Search Bar UI** – Command-style search interface

---

## 🖥️ Tech Stack

* HTML5
* CSS3 (Internal Styling)
* Vanilla JavaScript

---

## 🎯 Design Highlights

* Dark theme modern UI
* Gradient accents (purple/blue)
* Clean grid-based layouts
* Fully consistent design across pages
* Student-focused (not admin ERP)
